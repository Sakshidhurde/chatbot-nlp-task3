# 🤖 Chatbot using Hugging Face Transformers

## 📌 Project Description
This project is a console-based chatbot built using a pre-trained transformer model from Hugging Face. The chatbot can interact with users and generate human-like responses.

## 🎯 Objective
To build an interactive conversational chatbot using transformer-based NLP models.

## 🛠️ Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Jupyter Notebook / Google Colab

## ⚙️ Features
- Interactive chatbot
- Continuous conversation
- Dynamic response generation
- Exit condition (exit/quit)
- Uses pre-trained DialoGPT model

## 🔄 Workflow
User Input → Tokenization → Model Processing → Response Generation → Output → Loop

## ▶️ How to Run
1. Install required libraries:
   pip install transformers torch

2. Open the Jupyter Notebook

3. Run all cells

4. Start chatting with the bot
